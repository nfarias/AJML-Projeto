-- Projeto criado no Eclipse(Luna).
- Para ver o funcionamento por completo do sistema, será preciso um arquivo de texto.
- Dentro do projeto há dois arquivos, chamados "Arquivo.txt" e "Arquivo2.in" respectivamente.
- No Eclipse, após abrir e rodar o Main, insira o nome "Arquivo.txt" ou "Arquivo2.in" (sem aspas).
- A resposta será impressa no console do Eclipse para cada loop da geração da árvore e a árvore final,
armazenando esta última dentro da pasta do projeto com o nome "Resposta.txt".
